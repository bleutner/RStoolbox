Description

Intended for use with the Landsat 8 OLI QA band. Decodes pixel quality flags from integer to bit-words.

Usage

decodeQA(x)

Arguments

x

Integer (16bit)

See Also

encodeQA

Examples

decodeQA(53248)

#> [1] "1101000000000000"
